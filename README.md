# 🚀 KuaScore
> "Turn everyday transactions into opportunity."

**KuaScore** is a sleek, modern prototype of an AI-powered financial tool designed to transform informal digital behavior into a trusted financial identity. 

Millions of informal workers and small vendors rely heavily on mobile money transactions but remain invisible to traditional financial systems. KuaScore bridges this gap by providing an alternative credit scoring engine and a smart micro-investment bot to empower the underserved.

---

## ✨ Features

- 🧠 **Alternative Credit Scoring ("Trust Score")**: Analyzes mobile money transaction patterns to generate a dynamic, explainable Trust Score. It detects behavioral signals like income stability and repayment patterns to unlock financial opportunities.
- 💰 **Smart Savings Bot**: Automatically suggests flexible, safe daily saving amounts based on cash flow without disrupting the user's livelihood.
- 🔔 **AI Insights & Nudges**: Transparent notifications that explain how users can improve their score and alert them to upcoming financial needs.
- 📊 **Modern Fintech Dashboard**: A mobile-first, highly responsive UI built with an immersive dark-mode aesthetic, utilizing glassmorphism and modern visual storytelling.

---

## 🛠️ Technology Stack
- **Framework**: [Next.js](https://nextjs.org/) (React)
- **Styling**: Vanilla CSS (Global Variables, Utility Classes, Animations)
- **Architecture**: Mobile-first SPA approach with interactive React components.

## 📦 Installation & Setup

To run KuaScore locally on your machine, follow these steps:

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your machine (v18 or higher is recommended).

### 2. Clone the Repository
```bash
git clone <your-repo-url>
cd kuascore
```

### 3. Install Dependencies
```bash
npm install
```

### 4. Run the Development Server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the live prototype.

---

## 🌍 Deployment

This app is production-ready and optimized for deployment on Vercel.

1. Install the Vercel CLI globally (optional): `npm i -g vercel`
2. Run the deployment command inside the project directory:
   ```bash
   vercel
   ```
3. Follow the CLI prompts to instantly deploy your app.
